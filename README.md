# Power_BI-PhonePe-project
This dashboard provides a clear overview of PhonePe transaction trends, including total transactions, transaction value, user behavior, and regional performance. It delivers quick insights through interactive visuals to support data-driven decisions.

Create relationships between tables if needed.
